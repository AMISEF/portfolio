# CryptoSmart Hub — پنل مدیریت سرمایه

پلتفرم یکپارچهٔ مدیریت سرمایه و تحلیل بازار کریپتو (وب + مینی‌اپ تلگرام)، فارسی و راست‌چین.
دامنهٔ هدف: `portfolio.cryptosmart.site`

> **فاز ۱ (همین نسخه):** صفحهٔ خانه — نمای کلی بازار (Market Overview)

## ✨ امکانات فاز ۱

- 🎨 رابط راست‌چین فارسی، تم **روشن/تاریک** با نماد خورشید/ماه
- 📱 ناوبری موبایل (نوار پایین با دکمهٔ خانهٔ مرکزی) + دسکتاپ (نوار عمودی سمت راست)
- 🟦 طراحی سه‌بعدی شیشه‌ای با پالت رنگ برند (آبی مدرن + فیروزه‌ای)
- 🕒 تاریخ **شمسی و میلادی** + ساعت زندهٔ ثانیه‌شمار + چراغ‌های زندهٔ چشمک‌زن
- 📊 **نقشهٔ حرارتی بازار** (اندازه بر اساس ارزش بازار، رنگ بر اساس تغییر ۲۴ساعته)
- 😨 **شاخص ترس و طمع** به‌صورت گیج گرافیکی
- 🎞️ **نوار تیکر متحرک**: ارزش کل بازار، حجم، دامیننس بیت‌کوین/اتریوم/تتر، ارزش بازار اتریوم و آلت‌کوین‌ها
- 🚀 **بیشترین رشد توبیت** (۵ ارز با قیمت، رشد ۲۴ساعته و حجم)
- 💰 **قیمت‌های کلیدی**: تتر تومانی (تبدیل)، طلای ۱۸ عیار (سورس‌آرنا، هر ۳۰ دقیقه)، XAU/XAG/OIL از فیوچرز توبیت

## 🏗️ معماری

```
مرورگر / مینی‌اپ تلگرام
        │  (فقط با بک‌اند خودمان صحبت می‌کند)
        ▼
FastAPI (پایتون)  ──►  لایهٔ کش + بودجهٔ کردیت
        │
        ├─ CryptoRank  (کلان بازار + هیت‌مپ)  — کش ۱۰ دقیقه، سقف ۱۰هزار کردیت/ماه
        ├─ Toobit      (گینرها + فیوچرز فلزات/نفت) — کش ۱۲ ثانیه
        ├─ Tabdeal     (تتر تومانی) — کش ۱۵ ثانیه
        ├─ SourceArena (طلای ۱۸ عیار، پراکسی خارج) — کش ۳۰ دقیقه
        └─ alternative.me (ترس و طمع) — کش ۱۰ دقیقه
```

**کل منطق در پایتون** نوشته شده تا دیباگ و توسعه در فازهای بعد ساده باشد.
فرانت‌اند هیچ‌گاه مستقیم به APIهای بیرونی وصل نمی‌شود؛ همه‌چیز از طریق لایهٔ کش بک‌اند می‌آید.

### کنترل بودجهٔ کردیت CryptoRank
طبق پلن رایگان (۱۰٬۰۰۰/ماه، ۴۰۰/روز، ۱۰۰/دقیقه)، ماژول `CreditBudget` پیش از هر
فراخوانی سقف‌ها را بررسی می‌کند و در صورت رسیدن به سقف، دادهٔ کش‌شده را سرو می‌کند.
با TTL ده‌دقیقه‌ای، مصرف ماهانه حدود **۸٬۶۴۰ کردیت** (زیر سقف) باقی می‌ماند.

> اگر یک منبع در دسترس نباشد، سرویس به‌صورت خودکار دادهٔ کش کهنه یا **نمونهٔ واقع‌گرایانه**
> برمی‌گرداند و در UI با برچسب «نمونه/زنده» شفاف نمایش می‌دهد.

## 🚀 اجرا

```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env      # و کلیدهای واقعی را وارد کنید
python run.py             # یا: uvicorn app.main:app --reload
```

سپس به `http://localhost:8000` بروید.

## 📂 ساختار

```
app/
  main.py            # اپ FastAPI
  config.py          # تنظیمات + کلیدها از .env
  cache.py           # کش TTL + بودجهٔ کردیت
  routers/           # صفحات HTML + اندپوینت‌های JSON بازار
  services/          # CryptoRank, Toobit, Tabdeal, SourceArena, FNG + دادهٔ نمونه
  templates/         # قالب‌های Jinja2 (راست‌چین)
  static/css|js|img  # طراحی و منطق سمت کلاینت
```
