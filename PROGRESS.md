# وضعیت پیشرفت پروژه — CryptoSmart Hub

> خلاصهٔ کارهای انجام‌شده و گام بعدی. (به‌روزرسانی: شب کاری خودکار)

## ✅ انجام‌شده

### فاز ۱ — صفحهٔ خانه / نمای کلی بازار
- چیدمان سه‌ستونه (گینرها | هیت‌مپ + ترس‌وطمع | قیمت‌های کلیدی)، تیکر متحرک
- هیت‌مپ دسته‌بندی‌شده، گیج ترس و طمع زنده، آیکون ارزها، نماد تتر/طلا/نقره/نفت
- تم روشن/تاریک، راست‌چین، ناوبری موبایل/دسکتاپ
- اتصال APIها با کش و بودجهٔ کردیت + `/api/market/debug` برای عیب‌یابی

### فاز ۰ — احراز هویت و زیرساخت
- SQLite + SQLModel (`User`, `Holding`, `JournalEntry`)
- ثبت‌نام/ورود با pbkdf2، نشست کوکی امضاشده، منوی کاربر، خروج
- آمادهٔ ورود مینی‌اپ تلگرام (اعتبارسنجی initData) — نیازمند `TELEGRAM_BOT_TOKEN`

### فاز ۲ — داشبورد پورتفولیو
- CRUD دارایی‌ها، ارزش‌گذاری لحظه‌ای، سود/زیان، معادل تومانی
- نمودار تخصیص (دونات SVG) + جدول دارایی‌ها

### فاز ۳ (بخش عددی) — تحلیل ریسک سبد
- شاخص تمرکز (HHI)، سهم استیبل‌کوین، سطح ریسک، تغییر وزنی ۲۴ساعته
- بینش‌های قاعده‌محور فارسی (کارت «تحلیل ریسک سبد»)
- ⏳ لایهٔ تفسیر متن با هوش مصنوعی: نیازمند کلید مدل (Gemini/Dify)

### فاز ۴ — ژورنال معاملاتی
- CRUD معاملات + آمار سرور: نرخ برد، فاکتور سود، R:R، میانگین سود/زیان،
  بیشترین افت سرمایه، منحنی سرمایه (SVG)

## 🔧 تنظیمات مهم سرور (.env)
```
SECRET_KEY=<رشتهٔ تصادفی بلند>      # امضای کوکی نشست (ضروری در پروداکشن)
TELEGRAM_BOT_TOKEN=<توکن ربات>     # ورود خودکار در مینی‌اپ (اختیاری)
```
دیتابیس خودکار در `data/cryptosmart.db` ساخته می‌شود (gitignore).

## 🗺️ باقی‌مانده
- فاز ۳ کامل: تفسیر هوش مصنوعی تحلیل ریسک (کلید مدل لازم)
- فاز ۵: مینی‌اپ تلگرام + ربات اعلان
- فاز ۶: موتور هشدار (قیمت/شاخص)
- فاز ۷: درآمدزایی (اشتراک Pro/VIP، سطح‌بندی)
- همگام‌سازی پورتفولیو از صرافی (Read-Only)

## ⚠️ دقت قیمت‌ها
قیمت زنده فقط روی سرور (اینترنت باز) کار می‌کند. هر بخش برچسب «● زنده/● نمونه» دارد.
اگر «نمونه» دیدید: `curl -s https://portfolio.cryptosmart.site/api/market/debug` را بفرستید
تا نگاشت فیلدهای آن API را قطعی کنیم.
