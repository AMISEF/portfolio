# وضعیت پیشرفت پروژه — CryptoSmart Hub

> این فایل را برای دیدن خلاصهٔ کارهای انجام‌شده و گام بعدی بخوانید.

## ✅ انجام‌شده

### فاز ۱ — صفحهٔ خانه / نمای کلی بازار
- چیدمان سه‌ستونه (گینرها | هیت‌مپ + ترس‌وطمع | قیمت‌های کلیدی)
- هیت‌مپ دسته‌بندی‌شده، تیکر متحرک، گیج ترس و طمع، آیکون ارزها
- تم روشن/تاریک، راست‌چین، ناوبری موبایل/دسکتاپ
- اتصال APIها با کش و بودجهٔ کردیت + اندپوینت `/api/market/debug`

### فاز ۰ — احراز هویت و زیرساخت
- دیتابیس SQLite با SQLModel (`app/models.py`, `app/db.py`)
- ثبت‌نام/ورود با رمز (pbkdf2)، نشست کوکی امضاشده
- آمادهٔ **ورود مینی‌اپ تلگرام** (اعتبارسنجی initData) — فقط نیاز به `TELEGRAM_BOT_TOKEN`
- مودال ورود/ثبت‌نام، منوی کاربر، خروج

### فاز ۲ — داشبورد پورتفولیو
- افزودن/ویرایش/حذف دارایی (CRUD)
- ارزش‌گذاری لحظه‌ای، سود/زیان محقق‌نشده، معادل تومانی
- نمودار تخصیص (دونات SVG) + جدول دارایی‌ها
- صفحات: `/portfolio`, `/settings` (کاربردی)، `/journal` (پیش‌نمایش فاز ۴)

## 🔧 تنظیمات مهم سرور (.env)
به فایل `.env` روی سرور این‌ها را اضافه کنید:
```
SECRET_KEY=<یک رشتهٔ تصادفی بلند>          # برای امضای کوکی نشست (ضروری)
TELEGRAM_BOT_TOKEN=<توکن ربات>             # برای ورود خودکار در مینی‌اپ (اختیاری)
```
دیتابیس به‌صورت خودکار در `data/cryptosmart.db` ساخته می‌شود (در gitignore است).

## 🗺️ گام‌های بعدی (طبق پروپوزال)
- فاز ۳: لایهٔ هوش مصنوعی پورتفولیو (تحلیل ریسک، پیشنهاد سبدچینی) — نیازمند کلید مدل
- فاز ۴: ژورنال معاملاتی کامل + ایجنت تحلیل‌گر
- فاز ۵: مینی‌اپ تلگرام + ربات
- فاز ۶: موتور هشدار و اعلان
- فاز ۷: درآمدزایی (اشتراک، سطح‌بندی)

## ⚠️ نکتهٔ دقت قیمت‌ها
قیمت‌های زنده فقط روی سرور (با اینترنت باز) کار می‌کنند. اگر بخشی «نمونه» نشان داد،
خروجی `curl -s https://portfolio.cryptosmart.site/api/market/debug` را بفرستید تا نگاشت
فیلدهای آن API را دقیق کنیم.
